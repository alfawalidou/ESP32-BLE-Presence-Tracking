# Dashboard Guide

This repository includes a basic Node-RED starter flow.

## Open Node-RED

```text
http://SERVER_IP:1880
```

## Included Flow

The starter flow subscribes to:

```text
espresense/#
```

and sends all received messages to the debug sidebar.

## Next Dashboard Ideas

You can build:

- room status cards
- last seen device table
- distance graph
- person location dashboard
- alerts when a tag leaves/enters a room
- asset tracking dashboard
